libmd5wars - MD5大作战 Javascript 改版

------

Flash 游戏 [MD5大作战](https://deepmess.com/zh/md5.html) 的 Javascript 改版。

- 可在无头终端中运行
- 完全离线，无需网络连接
- 通过 API 接口操作（功能开发中）

拜谢游戏原作者 [Rinick](https://github.com/rinick)。
